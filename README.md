# SAEA

##File description

dataset : images to be attack
  
models : surrogate models
  
nets : surrogate model nets
  
SAEA-I-FGSM.py, SAEA-MI-FGSM.py, SAEA-TIM.py, SAEA-TI-DIM.py, SAEA-SI-TI-DIM.py : attack algorithm
  
## Requirements

Python 3.6.5

Tensorflow 1.12.0

Numpy 1.15.4

cv2 3.4.2

scipy 1.1.0
